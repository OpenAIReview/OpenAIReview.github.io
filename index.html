OpenAIReview
GitHub Documentation

AI-powered academic paper reviewer

Detect technical and logical errors in research papers using LLMs. Supports multiple review strategies, interactive visualization, and cost tracking.

$ pip install openaireview click to copy
1

Set your API key

echo "OPENROUTER_API_KEY=sk-..." > .env
2

Review a paper

openaireview review paper.pdf
3

Visualize results

openaireview serve
📄

Multiple Formats

PDF, DOCX, LaTeX, Markdown, and direct arXiv URL support.

🔎

Review Methods

Zero-shot, local deep-check, and incremental strategies for thorough analysis.

🤖

Model Flexibility

Works with any OpenRouter model. Built-in cost and token tracking.
